DiscreetOctupus: This Server and Website

This server was part of the Final Project for CSE 20289. My partners -- Stephanie Keene and Emma Ascolese -- and I were tasked with creating an HTTP server that supports directory listings, static files, and CGI scripts. That project, upon completion, was forked from its original GitLab project to serve as a basis for the server that is feeding your reqeusts right now. Minor modifications have been made (such as more intense error-checking), but the basis remains the same. You can view the current project by clicking the title above.
